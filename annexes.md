## 📖 Glossaire express

| Terme | Définition simple |
|---|---|
| **Algorithme** | Suite d'étapes précises pour résoudre un problème (une recette). |
| **Variable** | Boîte étiquetée qui contient une valeur. |
| **Fonction** | Bloc de code nommé, réutilisable. |
| **Compilateur** | Programme qui traduit du code (C) en binaire exécutable (`gcc`). |
| **Interpréteur** | Programme qui exécute du code ligne par ligne (Python). |
| **Pointeur** | Variable contenant l'adresse mémoire d'une autre variable (C). |
| **DOM** | Représentation de la page web manipulable par JavaScript. |
| **Front / Back** | Côté navigateur (visible) / côté serveur (caché). |
| **Serveur** | Ordinateur qui répond aux requêtes des clients. |
| **Base de données** | Système organisé pour stocker et retrouver des données (tables). |
| **Clé primaire / étrangère** | Identifiant unique d'une ligne / référence vers une autre table. |
| **SQL** | Langage pour interroger une base de données relationnelle. |
| **Modèle OSI** | Découpage en 7 couches de la communication réseau. |
| **Adresse IP** | Adresse d'une machine sur un réseau (`192.168.1.10`). |
| **Sous-réseau (subnet)** | Découpage d'un réseau en segments plus petits. |
| **DHCP / DNS** | Attribue une IP automatiquement / traduit un nom en IP. |
| **Hyperviseur** | Logiciel qui fait tourner des machines virtuelles (ESXi, Proxmox). |
| **Snapshot** | Photo restaurable de l'état d'une VM. |
| **SSH** | Connexion sécurisée à distance à un serveur. |
| **Hardening** | Durcissement : réduire la surface d'attaque d'un système. |
| **Chiffrement** | Rendre un message illisible puis lisible avec une clé (réversible). |
| **Hachage** | Empreinte irréversible (ex. SHA-256) ; pour les mots de passe. |
| **RGPD** | Loi européenne sur la protection des données personnelles. |
| **Git / GitHub** | Outil d'historique de code / site qui l'héberge en ligne. |

## 🆓 Plateformes d'entraînement gratuites, par domaine

**Programmation (Python, C)**
- [Exercism](https://exercism.org/) — exercices corrigés automatiquement, mentors bénévoles.
- [Codingame](https://www.codingame.com/) — apprendre en jouant.
- [France-IOI](http://www.france-ioi.org/) — progression algorithmique, en français, 100 % gratuit.
- [learn-c.org](https://www.learn-c.org/) — le C en interactif.

**Git**
- [Learn Git Branching](https://learngitbranching.js.org/?locale=fr_FR) — visuel, en français.

**Web (HTML/CSS/JS, PHP)**
- [freeCodeCamp](https://www.freecodecamp.org/) — parcours complets + certificats gratuits.
- [MDN Web Docs](https://developer.mozilla.org/fr/) — la référence du web, en français.
- [W3Schools](https://www.w3schools.com/) — tutoriels « try it yourself ».
- [Grafikart](https://grafikart.fr/) — tutoriels vidéo en français (PHP, web…).

**Bases de données (SQL)**
- [SQLBolt](https://sqlbolt.com/) — leçons SQL interactives.
- [W3Schools SQL](https://www.w3schools.com/sql/).

**Réseaux**
- [Cisco NetAcad](https://www.netacad.com/) — cours « Networking Basics » gratuits + badges officiels.
- [subnettingpractice.com](https://subnettingpractice.com/) — entraînement au subnetting.
- [Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer) — simulateur réseau gratuit.

**Systèmes & sécurité**
- [TryHackMe](https://tryhackme.com/) — rooms guidées (Linux/Windows Fundamentals…).
- [OverTheWire — Bandit](https://overthewire.org/wargames/bandit/) — jeu Linux, sans compte.
- [Root-Me](https://www.root-me.org/) — plateforme française gratuite, tous domaines.
- [Linux Survival](https://linuxsurvival.com/) — apprendre le terminal en douceur.

## 🎁 Pour aller plus loin (bonus, hors programme First Blood)

Ces sujets font partie de ta 1re année mais débordent du périmètre « prépa ». À explorer quand tu seras à l'aise sur les bases :

- **CCNA 2 (Cisco)** : la suite du réseau — *Switching, Routing and Wireless*. First Blood t'a donné l'avance sur le **CCNA 1** ; le 2 s'enchaînera naturellement en cours.
- **Unity & développement de jeu vidéo** : un module découverte du programme. Si la 3D/le jeu t'attire, le moteur **Unity** est gratuit, avec d'excellents tutoriels (Unity Learn).
- **Mathématiques pour l'informatique** : approfondis la logique, les bases de numération et l'arithmétique modulaire (les « Coins Maths » du programme en sont les graines).
- **Anglais technique** : continue à lire la doc en anglais — c'est le meilleur entraînement (les « English Corner » t'ont mis le pied à l'étrier).
